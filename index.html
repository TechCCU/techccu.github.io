簡介

TechCCU 是由中正大學資工系學生@brucehsu與@Terces所發起，我們秉持技術討論與經驗分享的精神連結校內外各式的人才與我們一起交流。 過去一年來我們採用各種方式，如線上分享、實體聚會（Meet up）的方式進行我們的活動，隨著我們人數的增加，我們聚集到更多願意一起分享 的人來參與，眼見時機成熟，我們決定主辦一場為期一天的 Workshop 來讓大家共襄盛舉。

講者

Richard Lee

Polydice & Inside 共同創辦人，目前主要參與公司產品愛料理的營運以及產品開發。 熱愛打造產品、新創公司的成長以及 Web 和 Mobile 相關技術。

我從資訊科系學生變成創業者學到的事

在這次的分享當中，我將以自身的經驗分享創業的經驗，以及作為資訊科系學生該如何在新創公司當中發揮價值，以及在學期間該如何提前規劃並學習相關知識。

Yungshin

上班爬網站，下班寫網站的程式小雜工，其實骨子裡是前端工程師，目前任職於資策會創研所

Introduction to Sails.js

Sails.js是一個Node.js MVC Framework，如同知名的Rails, Django等framework，Sails.js也支援現在Web開發中常見的feature，本次talk中將會介紹如何使用Sails.js及Sails.js的基礎架構

Kudo

中正資工系所土生土長的畢業生，曾任 CNA 會長，畢業後先後於趨勢科技與新創公司上班，目前任職新創公司 biideal 打雜技術長。目前努力學習成為一個 cloud/client 全端攻城獅。

flowstatd 之從學生到出社會的五年成長故事

學生時代總是很嘴砲地想要做很多大大小小的 project，最後很多都真的嘴砲化了，flowstatd 是唯一一個能出來見見世面的成果，也實際用於統計中正宿網流量。歷時五年（脫稿）打造，其中伴隨著個人從學生時代成長到出社會歷練後的不斷改造，本次將分享 flowstatd 的打造歷程，還有其中許多值得做為教材的不為人知面哦。

icemango

中正資工不知道哪年畢業的，最擅長把網頁上的字圈起來餵Google，目前在某家新創小公司跟同事們追求準時下班的小確幸。

OpenFlow introduction

以前擔任網管時總是希望能有更方便自動的網管系統，現在機會來了。在台灣剛開始發燒的軟體定義網路，踏入門的第 一把鑰匙，讓我們一起來推開 OpenFlow 的門扉 ...

Vexed

中正資工系、所畢。網路帳號 Vexed ，中正資工綽號米蟲。曾任職於中華電信 Xuite 、 Hami+ 。目前為 Yahoo 資深工程師。

網頁前端工程師與室內裝修師傅的相似之處

介紹我的網頁前端經歷與什麼是網頁前端工程師。

Luba Tang

長期關注 Open Source，專長是軟硬體界面整合，一不小心就跳坑到最深的編譯器與 EDA 領域。曾經開發商用 iterative compiler 與 open source MCLinker 專案，被 Android 與 BSD 所採用。現在專注在軟硬體效能提升上，為 Skymizer 創辦人。

Skymizer 雲端編譯 / Skymizer: Build Software Better and Faster On The Cloud

源碼的效能，如工程師的熱血，沒有極限，等待你來引爆。編譯器有許多的限制，壓抑了源碼的效能。從源碼結構、到執行時期、到硬體架構都有限制。本次主題會從 GCC 編譯器的限制出發，介紹 Skymizer 中新穎與重要的最佳化技術，如雲端編譯與 SkyPat 分析工具，如何突破這些限制來加速源碼。希望藉由這次的機會，分享過去開發編譯器的經驗，和與會的朋友們一起成長。

Terces

自中正資工系畢業，目前還在念中正資工所；平常為全端工程師(自稱)，長期關注硬體設計與開發，兼職於校內各個機房吹冷氣與掃地。

戰爭遊戲101

本Session為台灣資安發展協會贊助，希望透過有系統的教學方式，讓學員們可以了解War Game的精神以及闖關技巧，並學習以及實際操作駭客的攻擊手法，寓教於樂。
